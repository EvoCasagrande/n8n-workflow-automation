# n8n Automation Workflows

Portfolio of workflow automations built using n8n to automate business processes and integrate multiple tools.

These projects demonstrate how automation can eliminate repetitive tasks, connect different applications and improve operational efficiency.

## Projects Included

### Lead Capture Automation
Automatically captures leads from forms, stores them in Google Sheets and sends confirmation emails.

### WhatsApp Lead Notification
Sends instant notifications to businesses when a new lead is generated.

### PDF Data Extraction
Extracts relevant data from PDF documents received by email and stores the information in a spreadsheet.

### Automated Business Reporting
Collects data from multiple sources and sends automated reports via email.
